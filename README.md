# ordering-system
🔥 Adobo & Sarmale — Online Ordering System
PHP-based online food ordering web app for a Filipino-Romanian fusion restaurant with session-based cart, dynamic menu categorization, and toast notifications.

🛠️ Tech Stack

Backend: PHP (Sessions)
Frontend: HTML, CSS, Vanilla JavaScript
Database: (none / session-based)
Fonts: Google Fonts (Playfair Display, Instrument Sans, DM Mono)

✨ Features

Dynamic menu with category tabs (Starters, Mains, Desserts, Drinks)
Session-based cart with item count badge
Toast notifications on add to cart
Floating cart bar for quick checkout
Responsive layout for mobile and desktop
